# Book Shelf

JavaScript + Vite ile geliştirilmiş basit bir “Book Shelf” uygulaması.

Yazar: Canberk Yıldız

## Özellikler

- Kitapları ve kategorileri listeleme
- Detayları modal ile görüntüleme
- Alışveriş/okuma listesi yönetimi

## Kurulum

1. Bağımlılıkları yükle:

   `npm install`

2. Geliştirme modunu başlat:

   `npm run dev`

3. Üretim derlemesi al:

   `npm run build`
