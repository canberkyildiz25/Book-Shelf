# Book Shelf

A simple Book Shelf web app built with JavaScript + Vite.

Author: Canberk Yıldız

## Features

- Browse books and categories
- View book details in a modal
- Manage a shopping/reading list

## Getting started

1. Install dependencies

   `npm install`

2. Start dev server

   `npm run dev`

3. Build for production

   `npm run build`
